Um in 'git diff' immer einen Pager zu kriegen (also nicht einfach die Ausgabe ins Terminal geklatscht, was mir zu unübersichtlich ist), kann man folgende Einstellung setzen: 

$ git config --global pager.diff 'LESS=RS less -c'

Getestet mit git 1.7.4.1 aus Debian unstable (apt-get install git/unstable), mit älteren Versionen geht es anscheinend nicht.
